# JobMatch - Система подбора вакансий по навыкам

Веб-приложение для поиска вакансий на основе навыков с использованием семантического анализа. Пользователь вводит свои навыки или загружает резюме, а система находит наиболее подходящие вакансии с указанием процента совпадения, выявляет недостающие компетенции и даёт персонализированные рекомендации по профессиональному развитию. Приложение предназначено для двух категорий пользователей: соискателей, которые ищут работу, и работодателей, которые публикуют вакансии и управляют откликами.

## Функциональные возможности

### Для соискателей

- Регистрация и авторизация с хешированием паролей
- Загрузка резюме в форматах PDF и DOCX с автоматическим извлечением навыков через NLP-парсер
- Поиск вакансий по навыкам с расчётом процента совпадения
- Анализ недостающих навыков и генерация персональных рекомендаций по развитию
- Отклик на вакансию с сопроводительным письмом
- Отзыв отклика до его обработки работодателем
- Просмотр всех своих откликов с фильтрацией по статусам (на рассмотрении, принят, отклонён)

### Для работодателей

- Регистрация с указанием названия компании
- Публикация вакансий с полным описанием, зарплатой, локацией, требуемыми навыками и контактным телефоном
- Редактирование и удаление вакансий (удаление доступно только при отсутствии откликов)
- Просмотр откликов кандидатов с детальной информацией об их навыках и сопроводительных письмах
- Управление статусами откликов (принять или отклонить)

## Технологический стек

### Backend

- **Python** - язык программирования
- **Flask** - микрофреймворк для создания веб-приложений, обеспечивающий гибкую маршрутизацию и структуру проекта через механизм Blueprints
- **SQLAlchemy** - объектно-реляционное отображение для работы с базой данных через Python-объекты
- **SQLite3** - легковесная встраиваемая СУБД, не требующая отдельного сервера
- **Flask-Login** - управление сессиями пользователей
- **Flask-Bcrypt** - хеширование паролей с алгоритмом bcrypt
- **Flask-WTF** - создание и валидация форм, CSRF-защита

### Семантический анализ и обработка текста

- **Sentence Transformers** - библиотека для семантического анализа текста с использованием модели paraphrase-multilingual-MiniLM-L12-v2, которая преобразует текст в векторные представления (эмбеддинги) и вычисляет косинусное сходство
- **PyMorphy3** - морфологический анализатор для лемматизации (приведения слов к нормальной форме) русского языка
- **PyPDF2** - извлечение текста из PDF-файлов
- **python-docx** - извлечение текста из DOCX-файлов

### Frontend

- **Bootstrap 5** - CSS-фреймворк для адаптивной вёрстки и корректного отображения на любых устройствах
- **JavaScript** - клиентская валидация форм, подтверждение действий через модальные окна, автоматическое скрытие flash-сообщений, индикация загрузки
- **Jinja2** - шаблонизатор для наследования базового шаблона и динамической отрисовки содержимого

## Алгоритм сопоставления навыков

Процент совпадения вакансии и навыков пользователя рассчитывается по формуле:

**match_percentage = (skill_match × 0.7 + semantic_similarity × 0.2 + title_similarity × 0.1) × 100**

### Компоненты формулы

**skill_match (вес 70%)** - отношение количества совпавших навыков к общему числу требуемых навыков вакансии. Точное совпадение оценивается в 1 балл, частичное (когда один навык полностью входит в название другого) — в 0.5 балла. Для навыков «1с», «c#», «c++», «php» и «sql» установлен режим только точного совпадения для исключения ложных срабатываний.

**semantic_similarity (вес 20%)** - косинусное сходство между векторным представлением запроса пользователя и векторным представлением комбинированного текста вакансии (название + описание + навыки). Векторизация выполняется моделью Sentence Transformers paraphrase-multilingual-MiniLM-L12-v2, поддерживающей русский язык.

**title_similarity (вес 10%)** - косинусное сходство между запросом пользователя и названием вакансии.

### Предобработка текста

Перед расчётом производится очистка ввода от слов-паразитов («я умею», «владею», «знаю»), исправление частых опечаток («питон» → «python», «ворд» → «word», «ексель» → «excel»), приведение к нижнему регистру, удаление пунктуации и лемматизация через PyMorphy3.

## Установка и запуск

1. Клонировать репозиторий:
   ```
   git clone https://github.com/cajkalika39/job-selection-system.git
   cd job-selection-system
   ```

2. Создать виртуальное окружение:
   ```
   python -m venv .venv
   .venv\Scripts\activate (Windows)
   source .venv/bin/activate (Linux/Mac)
   ```

3. Установить зависимости:
   ```
   pip install -r requirements.txt
   ```

4. Запустить приложение:
   ```
   python app.py
   ```

5. Открыть в браузере: **http://127.0.0.1:5000**

При первом запуске автоматически создаётся база данных site.db и папка uploads для временного хранения загружаемых резюме.
